# aix-aleph-backend

Ein einfacher API-Server mit Node.js, Express und MongoDB (bereit für Erweiterung).

## 📦 Features

- 🌐 Express-Server mit JSON-API
- 🔐 .env-Unterstützung für Umgebungsvariablen
- 📥 POST-Route für Datenempfang (`/daten`)
- ✅ Einsatzbereit für REST-APIs und MongoDB-Anbindung
